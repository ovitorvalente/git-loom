# Code of Conduct

## Our Standard

Queremos que a comunidade do `gitloom` seja tecnica, respeitosa e produtiva.

Espera-se de participantes:

- respeito mutuo
- critica tecnica objetiva
- colaboracao de boa-fe
- foco em resolver problemas

## Comportamentos esperados

- discutir ideias sem ataques pessoais
- aceitar feedback e contestar com argumentos tecnicos
- manter conversas acessiveis e claras
- reportar problemas com contexto util

## Comportamentos inaceitaveis

- assedio ou intimidacao
- ataques pessoais
- linguagem discriminatoria
- spam, trolling ou flamewar
- exposicao publica irresponsavel de vulnerabilidades

## Escopo

Este codigo de conduta se aplica a:

- issues
- pull requests
- discussions
- reviews
- outros espacos publicos ligados ao projeto

## Enforcement

Maintainers podem remover, editar ou limitar participacoes que prejudiquem a comunidade ou o projeto.

## Como reportar

Se houver problema de conduta, reporte ao maintainer do projeto em canal apropriado, com contexto suficiente para analise.
